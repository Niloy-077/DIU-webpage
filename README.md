 DIU Practice Webpage
This repository contains my very first project learning raw web development. 
It is a practice webpage modeling a basic layout for Daffodil International University, built entirely from scratch based on a hand-drawn sketch.

* A clean, centered layout with a custom outer border.
* A structured HTML data table.
* A web form utilizing dropdown selection menus for course and language options.

Technologies Used
HTML for the structural foundation and content.
CSS for styling, alignment, and visual presentation.
